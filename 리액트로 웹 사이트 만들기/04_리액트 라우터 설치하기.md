# 04_리액트 라우터 설치하기 

#### ✅ 패키지 설치하기 

```bash
npm install react-router-dom@6
```

> `react-router-dom` : 패키지 이름 
>
> `@6` 패키지 버전을 6점대로 설치하겠다는 의미 



```json
  "dependencies": {
    "classnames": "^2.3.1",
    "react": "^17.0.2",
    "react-dom": "^17.0.2",
    "react-router-dom": "^6.12.1",
    "react-scripts": "4.0.3"
  },
```

> 설치가 정상적으로 되었음을 확인할 수 있다.
